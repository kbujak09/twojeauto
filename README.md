# **TwojeAuto** - 🚗 Aplikacja ogłoszeniowa

**TwojeAuto** to aplikacja webowa służąca do wystawiania ogłoszeń sprzedaży pojazdów.

🌍 **Live demo** - **https://twojeauto.vercel.app/**

## Funkcjonalności

- 📱 **Responsywność** - Aplikacja jest dostosowana pod każdy typ urządzeń

- 🛡️ **Bezpieczeństwo** - Autoryzacja użytkowników oparta na tokenach JWT oraz walidacja danych wejściowych za pomocą biblioteki Zod.

- ☁️ **Przechowywanie zdjęć w chmurze** - Zdjęcia są przesyłane do chmury w usłudze Cloudinary

- 🏃‍♂️‍➡️ **Dynamiczne filtrowanie** - Oferty są na bierząco aktualizowane przy ich filtrowaniu bez odświeżania strony

## 🛠️ Stack Technologiczny

### Frontend
* **React + Vite**
* **TypeScript**
* **TanStack Router**
* **SCSS**

### Backend
* **Node.js + Express**
* **MongoDB + Mongoose**
* **Zod**
* **JWT**

### Infrastruktura i zewnętrzne usługi
* **Vercel** (Hosting aplikacji webowej i API)
* **MongoDB Atlas** (Baza danych w chmurze)
* **Cloudinary** (Serwer plików graficznych)
